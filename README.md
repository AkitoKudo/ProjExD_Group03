# こうかとんシューティング(仮)

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
縦か横スクロールゲーム


## ゲームの遊び方
* 十字キーで移動
* 特定キーでビームを放つ
* 敵を倒してスコアを稼ぐ
* 残機が0になったらゲームオーバー
* スコアを消費して特殊機能が発動可能
* 一定スコアでボスが出現


## ゲームの実装
### 共通基本機能
* 背景画像とキャラクターの描画
* スコア表示
* 弾数表示
* 横スクロール

### 担当追加機能
* 残機(くどう):残機の追加、被弾した際残機を減らしシールドを獲得。最大残機３
* 障害物(みき)
* アイテム(ひらなべ)
* 敵のレベルアップ(しのはら)
* クリア画面,ラスボス作成(きざき)
